[BvS Encryption & Decryption Tool]
01_ Project Plan
Posted on 11 June 2019

The FUI shown in the movie BvS is a mixture of different programs. The middle container is the decryption program. The surrounding programs perform other actions unrelated to encryption or decryption. Therefore our program will not have as many tools displayed on the screen.

The program should display the following items:

  • Input form to choose file to encrypt/decrypt
  • Encrypted/Decrypted data file location
  • Drive letter for Encrypted/Decrypted data file location
  • Encryption/Decryption progress percentage
  • Encryption/Decryption progress bar
  • Encryption/Decryption runtime
  • Estimated time till completion
  • Section stating current process
  • Pop-Up completion message
  • File Report after completion
